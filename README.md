# 及時天氣與華氏攝氏溫度轉換器
- 使用者可以查看臺北與Buffalo及時天氣狀況
- 使用者可以輸入華氏或是攝氏溫度
- 自動溫度轉換
---
- 使用 create react app 建立本專案
- 使用 gh-pages 套件包將專案發布到 github pages上
- 歡迎試用本專案: https://vanessalin9.github.io/temperature-P06/
